SHOPPING APPLICATION (E-COMMERCE)

Java, Hibernate, Servlet, HTML, CSS, JSP, and JPQL


Description: Developed a full-stack e-commerce application using Java, Hibernate, Servlet, HTML, CSS, JSP, and JPQL. The application consisted of three main modules: Admin, Merchant, and Customer. Key Features:



1. Admin Module: • Implemented an admin dashboard for managing the entire system. • Provided functionalities to Approve, Delete, Block merchants and customers. • Created user management, including registration, login, and authorization.


2. Merchant Module: • Built a dedicated module for merchants to manage their products and inventory. • Allowed merchants to add new products, update prices, and track inventory levels.


3. Customer Module: • Created an interactive user interface for customers to browse and purchase products. • Implemented a search functionality to allow customers to find products based on Name or categories. • Enabled customers to add items to their shopping cart, proceed to checkout.


➢ Technologies Used: • Java: Utilized the Java programming language for developing the application’s backend logic and business rules. • Hibernate: Incorporated Hibernate as the Object-Relational Mapping (ORM) tool for database operations and persistence. • Servlet: Servlets used for handling HTTP requests and responses, managing session data, and implementing business logic. • HTML and CSS: Established the user interface using HTML for structure and CSS for styling and layout. • JSP: Used JSP (Java Server Pages) for server-side rendering of web content. • JPQL: Made use of JPQL (Java Persistence Query Language) for writing database queries and retrieving data.


This project showcased my skills in developing a full-stack application using Java and various web technologies. It demonstrated my proficiency in back-end development, front-end design, database management, and integration of different modules.
